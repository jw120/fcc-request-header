FreeCodeCamp Back End Exercise: Request Header Microservice

Microservice API which extracts some basic information from a GET request's headers and returns as JSON.

Example

/api/whoami returns 

{"ipaddress":"::1","language":"en-US","software":"Macintosh; Intel Mac OS X 10_11_5"}
(values depend on your location and browser)
